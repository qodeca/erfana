# Erfana v0.11.2

_Released: 2026-06-01_

- Copy, cut and paste work reliably again across the editor, terminal, dialogs and preview – fixing a sandbox bug that silently broke the clipboard.
- The macOS app is much smaller to download (~1.0 GB → ~610 MB), with no features removed.
- Streamlined distribution: macOS ships a single Apple Silicon build, Windows a single installer. (Intel Macs and Linux no longer have downloads.)
- Security: patched high-severity issues in two bundled libraries (axios, fast-uri).
