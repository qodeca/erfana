# Erfana v0.10.0
_Released: 2026-05-31_

- In-app AI prompts (Modify, Visualize, and the diagram tools) now apply their changes directly to your document instead of printing to the terminal.
- New terminal maximize — expand the terminal over the editor with Cmd/Ctrl+Shift+M for full-screen terminal work.
- Video transcription now works in installed builds (per-architecture ffmpeg), with quieter, more accurate logging.
