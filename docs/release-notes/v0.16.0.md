# Erfana v0.16.0

_Released: 2026-06-14_

- The Claude Code context status bar (model + context-usage meter in the terminal) now works on Windows, not just macOS.
- Refreshed home screen: a new background image with Import and Recent Projects grouped in a tidy bottom-right panel and the version in the heading.
- Fixed: in Preview, "Modify" and "Ask" now act on the text you selected in documents with frontmatter, instead of grabbing the frontmatter.
- Smoother editor/preview scroll alignment in files that have frontmatter.
- A harmless error message that could appear when quitting is gone.
