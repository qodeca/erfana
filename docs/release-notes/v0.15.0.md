# Erfana v0.15.0

_Released: 2026-06-09_

- **Safer multi-window editing** — when you open the same project in two Erfana windows, the behind-the-scenes lock that coordinates them is now signed and self-healing: a crashed or force-quit window no longer leaves the project blocked (its lock is reclaimed automatically), locks survive sleep/wake, and a stale lock from another process on your machine can't hijack the project.
- **Text is selectable again across the app** — you can copy error messages, file paths, status data, dialog and toast text, and settings descriptions, and the markdown-preview AI context menu (Explain / Modify / Ask / Visualize) works again ([#211](https://github.com/qodeca/erfana/issues/211)).
