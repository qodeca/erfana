# Erfana v0.9.5
_Released: 2026-04-25_

- Erfana releases are now cryptographically signed and notarized on macOS (Developer ID + Apple notarization), Windows (Azure Trusted Signing certificate), and Linux (minisign).
- All releases are accompanied by a minisign-signed SHA256SUMS file — operators can independently verify every binary against the published Ed25519 public key.
- Windows code signing migrated from OIDC to certificate-based authentication for Azure Trusted Signing (electron-builder 26 compatibility).
- macOS notarization now retries the stapling step against Apple's ticket-database lag, fixing a class of first-launch "app cannot be opened" failures.
- Multi-reviewer audit closed all reviewer findings on the new release pipeline; v0.9.5 is the first release built by the production version of that pipeline.
