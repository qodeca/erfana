# Erfana v0.12.0

_Released: 2026-06-04_

- **Screenshot capture now works on Windows and Linux.** The terminal's screenshot tool was previously macOS-only. You can now grab a full screen, pick a specific window from a thumbnail picker, or drag to select a region — the image path drops straight into the terminal.
- **Text selection works again in the markdown preview.** You can select and copy rendered preview text normally.
- **Stability fix for very large projects.** Removed a file-watcher resource leak that could crash PDF/DOCX export in folders containing many thousands of files.
