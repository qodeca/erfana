# Erfana v0.14.0

_Released: 2026-06-06_

- **Consistent terminal font across platforms** — Erfana now bundles the Cascadia Mono font, so the Windows terminal matches the polished macOS look and renders identically everywhere.
- **Window title shows your project and version** — reads `ERFANA v0.14.0` with no project open, or `{Project} | ERFANA v0.14.0` with one open, on both Windows and macOS.
- **More reliable git status on Windows** — fixed false "modified" indicators caused by line-ending (CRLF) handling, and better detection when a folder becomes or stops being a git repo.
- **Accurate Claude Code context bar on launch** — fixed a bug where a freshly launched session could show a previous session's context percentage.
- **Project sidebar shows the folder name on Windows** — no longer displays the full `C:\…` path.
