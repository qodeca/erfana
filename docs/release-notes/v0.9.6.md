# Erfana v0.9.6

_Released: 2026-05-22_

- **Terminal restored on macOS.** v0.9.5 macOS builds shipped a broken terminal — every command returned `Error: posix_spawnp failed.` because the bundled spawn-helper lost its executable bit during signing. Anyone on v0.9.5 macOS must upgrade.
- **License clarified.** Erfana is now formally licensed under a proprietary "All rights reserved" agreement held by Qodeca sp. z o.o. No change in how the app is distributed; the temporary MIT placeholder added during release prep has been replaced.
- **Security update.** Bundled `tar` upgraded to 7.5.11 (previous 7.4.0 was npm-deprecated with security advisories). Internal trust chain only — no user-visible behaviour change.
