# Erfana v0.13.0

_Released: 2026-06-05_

- **See your Claude Code context usage in the terminal (macOS).** When you run Claude Code in a terminal panel, a thin status bar shows the model (e.g. "Opus 4.8"), the context-window size, and how much you've used — with a progress bar that shifts green → orange → red as it fills. Reads only from Claude Code's own session files; never changes your config.
- **Reliable build on hardened Windows 11.** A fresh install now rebuilds the terminal backend successfully on locked-down Windows 11 machines, plus small editor- and screenshot-cursor fixes.
- **Filenames stay out of log files.** Invalid file/folder names you type are still shown on screen, but local log files now record `[redacted-filename]` instead of the name.
